# Ice Cream Orders
This is an application that lets one make an order of icecream

## Installation and Running

### Requirements
- Node should be installed 
- A package manager e.g. yarn or npm

### Clone the repository
```
$ While in the directory you want to clone the project forexample `/Desktop`
$ git clone https://github.com/EleisonC/Ice.git
$ cd Ice in the directory of the package.json
```

## Installing dependancies

- If you have yarn installed
```
$ yarn install
```

- If you are using npm
```
$ npm install
```

## Running the application

- If you have yarn installed
```
$ yarn start
```

- If you are using npm
```
$ npm start
```
